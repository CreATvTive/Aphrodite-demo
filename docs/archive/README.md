# Aphrodite documentation archive

All directories below are historical, legacy, superseded, experimental, or documentation-only. Nothing under `docs/archive/` defines current chatbox v0 runtime, persona, contracts, field semantics, language conditions, embodiment mapping, or acceptance. Current authority starts at [`../chatbox/phase-plan-v0.md`](../chatbox/phase-plan-v0.md).
